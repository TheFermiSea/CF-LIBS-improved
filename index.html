CF-LIBS Documentation

CF-LIBS is a physics-based Python library for forward modeling and analysis of LIBS plasmas. The shipped algorithm is physics-only — see Evolution Framework for the constraint spec.

Inversion Config

The classic CF-LIBS inversion uses an analysis section in your config file to control line detection, selection, and solver settings. See CF-LIBS User Guide for the full schema and CF-LIBS API Reference for CLI flags.
