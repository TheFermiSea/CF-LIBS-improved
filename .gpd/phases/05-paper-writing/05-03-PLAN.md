  <name>Task 1: Cross-check numerical claims and figure-text consistency</name>
  <files>.gpd/phases/05-paper-writing/consistency_check.md</files>
  <action>
    Perform a systematic cross-check of every numerical claim in the manuscript against source data.

    **Step 1: Extract all numbers from results.tex**
    Search for every numerical value in paper/sections/results.tex (speedup factors, error values, throughput numbers, batch sizes, test counts). List each with its line number and the claim it supports.

    **Step 2: Verify against source data**
    For each number, verify against:
    - benchmarks/figures/benchmark_summary.json (headline_numbers and paper_claims)
    - validation/accuracy/results/accuracy_report.json (if it exists)
    - .gpd/phases/04-validation-accuracy/04-01-SUMMARY.md (accuracy table)
    - .gpd/phases/04-validation-accuracy/04-02-SUMMARY.md (real-data results)

    **Step 3: Check abstract numbers**
    Verify every number in paper/sections/abstract.tex matches the results section and source data.

    **Step 4: Figure-text consistency**
    For each figure reference in the text:
    - Verify the PDF file exists in paper/figures/
    - Verify the textual description matches what the figure shows
    - Verify the figure label matches the \ref{} call

    **Step 5: Bibliography completeness**
    - Check every \cite{} command has a matching refs.bib entry
    - Check every refs.bib entry is cited at least once
    - Verify all 4 must-surface references are cited: ExoJAX, HELIOS-K, Zaghloul 2024, Evans 2018

    **Step 6: Notation consistency**
    Spot-check 10 equations for notation consistency:
    - gamma always means Lorentzian HWHM
    - sigma always means Gaussian std dev
    - T in eV (or explicitly marked Kelvin)
    - n_e in cm^-3
    - lambda in nm

    Write all findings to .gpd/phases/05-paper-writing/consistency_check.md with:
    - Section: "Numerical Claims" -- table of [claim, text value, source value, match Y/N]
    - Section: "Figure-Text" -- table of [figure, ref exists, PDF exists, description matches]
    - Section: "Bibliography" -- list of any missing or orphaned references
    - Section: "Notation" -- any inconsistencies found
    - Section: "Issues Found" -- list of discrepancies requiring fixes
  </action>
  <verify>
    1. consistency_check.md exists and contains all 5 sections
    2. Every number in results.tex is listed in the numerical claims table
    3. Every figure reference is verified
    4. All 4 must-surface references confirmed cited
    5. Any discrepancies are listed with specific line numbers and corrections needed
  </verify>
