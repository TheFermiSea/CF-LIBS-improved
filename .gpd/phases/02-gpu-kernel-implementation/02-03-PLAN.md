    Create cflibs/manifold/batch_forward.py implementing the batch forward model from DERV-05.

    The forward model composes four stages per DERV-05 Eq. (01-03.4):

    **Stage 1 -- Saha ionization balance:**
    Given T_eV, n_e, and compositions C_i, compute ionization fractions f_z for each element.
    Use the same Saha physics as anderson_solver.py (or a simplified direct computation if Anderson iteration is too expensive per-spectrum for the batch case -- see note below).

    NOTE: For the batch forward model used in manifold generation, the caller typically provides (T_eV, n_e) directly (they are grid points, not being solved for). In this case, Saha balance gives ionization fractions GIVEN n_e, without needing to iterate for self-consistent n_e. This is a FORWARD model, not an inversion. The Anderson solver (from Plan 02-02) is for the INVERSION case where n_e is unknown. The batch forward model should:
    - Accept (T_eV, n_e, C) as given parameters
    - Compute Saha ratios S = (2*U_{z+1})/(n_e*U_z) * SAHA_CONST_CM3 * T_eV^1.5 * exp(-chi/T_eV)
    - Derive ionization fractions: f_I = 1/(1+S1+S1*S2), f_II = S1*f_I, f_III = S2*f_II
    - This is a DIRECT computation (no iteration), making it cheap per spectrum

    **Stage 2 -- Boltzmann level populations:**
    n_k = C_s * n_e * f_z * (g_k / U_z) * exp(-E_k / T_eV) [cm^-3]
    (per element s, ionization stage z, upper level k)

    **Stage 3 -- Line emissivities:**
    epsilon_l = (hc / (4*pi*lambda_l)) * A_ki * n_k * 1e6  [W m^-3 sr^-1]
    The 1e6 converts cm^-3 -> m^-3 per locked convention.

    **Stage 4 -- Voigt broadening + assembly:**
    Use voigt_spectrum_jax from profiles.py (Plan 02-01):
    S(lambda) = sum_l epsilon_l * V_l(lambda)
    where V_l is the Voigt profile for line l with per-line sigma and gamma.
