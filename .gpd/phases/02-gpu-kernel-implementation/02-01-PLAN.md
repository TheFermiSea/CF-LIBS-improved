  <name>Task 2: Implement batched Boltzmann WLS kernel (IMPL-02)</name>
  <files>cflibs/inversion/boltzmann_jax.py, tests/test_boltzmann_jax.py</files>
  <action>
    Create a new module cflibs/inversion/boltzmann_jax.py implementing the batched weighted least-squares Boltzmann fitting from DERV-02 (Eq. 01-01.4).

    1. Implement `batched_boltzmann_fit(x, y, w, mask)`:
       - x: (B, N_max) upper-level energies E_k [eV]
       - y: (B, N_max) Boltzmann plot values ln(I*lambda/(g_k*A_ki)) [dimensionless]
       - w: (B, N_max) weights (1/sigma_y^2) [dimensionless]
       - mask: (B, N_max) boolean mask for valid lines
       - Returns: BoltzmannFitResult dataclass with slope (B,), intercept (B,), T_K (B,), sigma_T (B,), R_squared (B,), n_valid (B,)

    2. The closed-form WLS uses 5 dot products per DERV-02:
       w_masked = w * mask  # zero out padded entries
       S_w = sum(w_masked, axis=1)
       S_wx = sum(w_masked * x, axis=1)
       S_wy = sum(w_masked * y, axis=1)
       S_wxx = sum(w_masked * x * x, axis=1)
       S_wxy = sum(w_masked * x * y, axis=1)
       det = S_w * S_wxx - S_wx**2
       slope = (S_w * S_wxy - S_wx * S_wy) / det
       intercept = (S_wxx * S_wy - S_wx * S_wxy) / det
       T_K = -1.0 / (slope * KB_EV)  # KB_EV = 8.617333e-5 eV/K

    3. Handle degenerate cases:
       - det ~= 0: set slope=0, intercept=0, T_K=0 (flagged via n_valid)
       - Use jnp.where(jnp.abs(det) > 1e-30, ..., 0.0) for safe division

    4. Compute uncertainty from covariance:
       sigma_b = sqrt(S_w / det)  # std error of slope
       sigma_T = T_K^2 * KB_EV * sigma_b  # error propagation

    5. Compute R-squared:
       y_pred = intercept[:, None] + slope[:, None] * x  # (B, N_max)
       SS_res = sum(w_masked * (y - y_pred)**2, axis=1)
       y_mean = S_wy / S_w
       SS_tot = sum(w_masked * (y - y_mean[:, None])**2, axis=1)
       R_squared = 1 - SS_res / SS_tot

    6. Decorate with @jit. All operations are pure JAX array ops, no Python loops.
       # ASSERT_CONVENTION: x = E_k [eV], y = ln(I*lambda/(g_k*A_ki)) [dimensionless], slope [eV^-1]

    7. Also provide a convenience function `boltzmann_temperature_jax(x, y, w, mask)` that returns only T_K and sigma_T for simpler use cases.

    8. Write test file tests/test_boltzmann_jax.py with:
       a. test_known_temperature: Synthetic data at T=10000K, recover T within sigma_T
       b. test_batch_consistency: B=10 independent fits, compare each against individual np.polyfit
       c. test_unweighted_reduction: w=1 recovers OLS (compare against np.polyfit)
       d. test_single_element_batch: B=1, verify no shape issues
       e. test_degenerate_det: All E_k identical, verify det~0 handling (no NaN/Inf)
       f. test_two_lines_exact: N=2, verify exact fit (R^2=1)
       g. test_masking: Mixed N_valid per batch element, verify correct results
       h. test_accuracy_vs_numpy: 100 random batch elements, max relative error < 1e-10 vs numpy
       i. test_gradient: jax.grad through batched_boltzmann_fit produces finite values
       j. test_temperature_uncertainty: Verify sigma_T is reasonable (within 10% of Monte Carlo estimate)
       All tests marked with @pytest.mark.requires_jax and @pytest.mark.unit.

    Do NOT modify the existing boltzmann.py module. The JAX kernel is a parallel implementation, not a replacement.
  </action>
